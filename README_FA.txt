📘 راهنمای اجرای ربات تلگرام سیگنال‌دهی Pocket Option:

۱. یک حساب کاربری در سایت render.com بسازید.
۲. پروژه جدید ایجاد کنید (New Web Service).
۳. سورس کد این ربات را از فایل ZIP آپلود کنید.
۴. فایل render.yaml به صورت خودکار تنظیمات اجرا را انجام می‌دهد.
۵. متغیرهای محیطی (Environment Variables) زیر را وارد کنید:
   - BOT_TOKEN: توکن ربات شما
   - USER_ID: آیدی عددی تلگرام شما (نه @username بلکه ID عددی)
۶. دکمه Deploy را بزنید.

✅ بعد از راه‌اندازی، ربات فقط به شما پیام می‌فرستد.

📌 توجه: برای دریافت سیگنال‌های واقعی باید اندیکاتورها را در تابع send_signal پیاده‌سازی کنید.
